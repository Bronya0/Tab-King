主要功能：
- 多搜索引擎支持：Google、Bing、百度切换
- 自定义搜索建议服务器：可选择不同的建议源或使用自定义API
- 实用工具集成：Base64编码解码、JSON格式化、密码生成器、时间戳转换、颜色选择器、二维码生成等
- 个性化设置：支持自定义背景、布局调整、数据备份恢复
- 快捷方式管理：可自定义添加和管理常用网站
- 现代化UI设计：毛玻璃效果、流畅动画、响应式布局

高级功能：
- 支持自定义搜索建议URL，使用{query}作为搜索词占位符
- 支持多种编码格式和进制转换
- 图片压缩和文件哈希计算
- Unicode编码解码和单位换算
- 图片压缩、文件哈希小工具等等

使用场景：
- 替代浏览器默认新标签页
- 开发者工具集合

Tab King让您的浏览器新标签页更加强大、美观和实用！
